# KaraokeStar MVP Plan

- Objective: Build a location-based karaoke discovery and queueing app with tipping, favorites, and basic payout scaffolding.
- MVP Scope:
  - Discover nearby karaoke events based on location
  - Join queue and tip KJs to influence standing
  - Collect and utilize user favorites for recommendations
  - Basic payout scaffolding (Stripe) without full onboarding
  - Simple UI for venues/KJs to join and manage queues (in MVP scope as admin views)
- MVP Deliverables:
  - MVP plan (exists)
  - MVP README
  - starter YAML/JSON model for events, KJs, queues, tips, and favorites
  - Minimal mobile UI scaffold (Flutter) or a simple web dashboard UI
- Milestones:
  - Week 1: Data model design and local prototype; YAML/JSON storage
  - Week 2: Location-based discovery and queue simulation; tipping feature
  - Week 3: Favorites and basic recommendations
  - Week 4: Stripe payout scaffolding and admin interfaces
